Quick Start Guide for Windows Machines

  • Follow instructions on Canvas for installing SPARK Ada on your own machine and accessing SPARK Ada on Swanse University computers.
  • Then follow the Instructions for Downloading the Git Repository for CSC313/CSCM13 (this may have happened already if you followed the Video guide)
  • Test your setup by using the exampleCodeForTesting the installation found in the git repository your installation
  • Study Basic Usage Instructions for SPARK Ada
  • Check out SPARK Ada code examples (including Examples from the lectures and other sources)
    Especially we recommmend to study the "Examples for Getting Started".
    The example
    examplesForCoursework/conversion24hrClock12hrAMPMClock
    can be used as a good starting point for answering Question 1 of the coursework
  • The lab sheet and the coursework can be found under Canvas.
  • On Canvas you can find as well a useful cheat sheet for SPARK Ada.

Instructions for Linux Machines

  • Start by following the Instructions so that you can use SPARK Ada and why3 from the File Manager
  • Copy the folders
    • ~/git/criticalhighintegritysystems/exampleCodeForTesting/
    • ~/git/criticalhighintegritysystems/sparkAdaCodeLecture/
    to a location where you want to work with
    • Reason is: Git repository updates frequently
    • If you work on it, updating of git repository failes due to conflict.
    • Should you ever get problems, move
      ~/git/criticalhighintegritysystems
      to a new destination and set up git repository again following Instructions for downloading Git repository with SPARK Ada code examples
  • Some basic instructions about using Linux and Emacs can be found here
  • Instructions for setting up machines in the Linux Lab for using SPARK Ada and why3 from command line
