Downloading Examples from Lecture CSCM13/CSC313 in Linux Lab

  • If you followed the initial instructions for setting up Spark Ada in the Linux Lab, the examples from the lectures should be at
    home directory -> git -> criticalhighintegritysystems
  • Otherwise you can find the examples from the lectures and other sources in
    this git repository
    • To download to a Linux machine execute from the command prompt (requires installation of git)
      git clone https://anton_setzer@bitbucket.org/anton_setzer/criticalhighintegritysystems.git
    • Please store the files in the directory ~/git/criticalhighintegritysystems

Make a copy of the git respository (Linux Machines in Linux Lab)

  • It is highly highly recommended to make a copy of files from the git repository and work on those copies
  • Otherwise you might not be able to update the git repository because of conflicing files created by yourself.
  • Should this have happened, move your directory git/criticalhighintegritysystems to somewhere else and download the git repository again.

Updating Git Repository (Linux Machines in Linux Lab)

  • The standard script for installing SPARK Ada in the Linux lab will install a command updateGitCritsys in your directory bin
  • Execute it from Console to update your git repostory
  • You can also update it manually from your home directory by running in git/criticalhighintegritysystems the command git clone
