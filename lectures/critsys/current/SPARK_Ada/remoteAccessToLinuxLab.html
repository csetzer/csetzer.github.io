Remote Access to Linux Lab

  • The following has not been tested yet. Please test it before relying it.
  • Note that there is no computer support available Friday late afternoon - Monday morning.
  • From Linux Machines you can get remote login to any other linux machine by using the command

    ssh -X username@machineaddress

  • The user name is your login name.
  • The machine address should be

    machinename.swan.ac.uk

    where machinename is the machine name of a machine in the linux machine (displayed at login).
  • Note that if too many students access the same machine it might get overloaded. So it is good to use different machines
